# IntelliCore Frontend

This is the React-based frontend for the IntelliCore Enterprise AI HR KMS.

## ✨ Features

- **Dynamic Dashboards**: Custom views for Employees and HR Administrators.
- **Intelligent Chat Interface**: Interactive chat with policy retrieval and tool tracing.
- **Document Management**: Seamless upload and management of HR policy documents.
- **Real-time Analytics**: Visual representation of system metrics and policy gaps.
- **Responsive Design**: Built with Tailwind CSS for a premium look on all devices.

## 🚀 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: React Hooks & Context API
- **Charts**: Recharts

## 🛠️ Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure the API URL in `src/config.js` (if applicable).
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main application views (Login, Chat, Admin, Analytics).
- `src/services`: API interaction logic.
- `src/assets`: Static assets and global styles.
